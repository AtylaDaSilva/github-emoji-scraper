# GitHub Emoji Scraper

A scraper designed to fetch emojis from the [GitHub Emoji Cheat-Sheet](https://github.com/ikatyang/emoji-cheat-sheet/blob/master/README.md).

## How to use

1. Make sure you have the latest version of [Node.js](https://nodejs.org/en/)
2. Clone this repository into a directory of your choosing.
3. With a Command Line Interface, `cd` into the repository you just cloned.
4. Run the command `npm run start` and wait for the process to finish.
5. The emojis will be stored in a JSON file and will be saved in the directory `./out/emojis_timestamp.json`.

## Dependencies

* [Node.js](https://nodejs.org/en/)
* [Puppeteer](https://pptr.dev/)
